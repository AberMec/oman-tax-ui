# 🇴🇲 Oman Tax Compliance UI

## 📌 Overview
This project is a **frontend-only demonstration** of the Oman Tax Compliance Service.

It provides a simple and clean user interface to simulate tax calculation workflows while maintaining a strict separation from the backend system.

> ⚠️ The backend API and database are private and not exposed in this repository.

---

## 🎯 Purpose
The goal of this project is to:
- Demonstrate frontend interaction for tax calculation
- Showcase clean UI structure and user flow
- Provide a safe public demo without exposing backend logic or credentials

---

## ⚙️ Features
- Income-based tax calculation (demo simulation)
- Clean and responsive UI
- Lightweight and fast (no frameworks required)
- Clear separation between frontend and backend

---
